# **Army Minigame on JS**

## **Overview**  
The **Army Minigame on JS** project includes various small games developed using **JavaScript**. It features games like **Raft Game** and **Snake Game**, created during military service.

## **Features**  
- **Raft Game**: A survival game where you control a raft and avoid obstacles.  
- **Snake Game**: A classic snake game where you guide the snake to grow longer by eating food.  
- **Minimalistic Design**: Simple game mechanics with **JavaScript**, **HTML**, and **CSS**.

## **How to Play**  
1. **Launch**: Open `index.html` in a browser.  
2. **Navigate**: Use keyboard or mouse to control.  
3. **Survive & Score**: Try to beat your own score by surviving as long as possible.

## **Project Structure**  
- `index.html`: Main HTML file to start the games.  
- `style.css`: The styling for the games.  
- `goorm.manifest`: Configuration file for the project.  
- `Raft_Game/`, `Snake_Game/`: Folders for each game's assets.

### **한국어**

# **Army Minigame on JS**

## **개요**  
**Army Minigame on JS** 프로젝트는 **JavaScript**로 개발된 다양한 작은 게임들을 포함하고 있습니다. **Raft Game**과 **Snake Game** 같은 게임들이 군 복무 기간 중 만들어졌습니다.

## **주요 기능**  
- **Raft Game**: 뗏목을 조종하며 장애물을 피하는 생존 게임입니다.  
- **Snake Game**: 음식을 먹고 길어지는 뱀을 조종하는 고전적인 뱀 게임입니다.  
- **미니멀한 디자인**: **JavaScript**, **HTML**, **CSS**를 활용한 간단한 게임 메커니즘입니다.

## **게임 방법**  
1. **게임 시작**: `index.html` 파일을 브라우저에서 엽니다.  
2. **조작**: 키보드나 마우스를 사용하여 게임을 조종합니다.  
3. **생존 & 점수**: 가능한 오래 생존하여 높은 점수를 기록합니다.

## **프로젝트 구조**  
- `index.html`: 게임을 시작하는 주요 HTML 파일입니다.  
- `style.css`: 게임의 스타일링을 담당하는 CSS 파일입니다.  
- `goorm.manifest`: 프로젝트의 구성 파일입니다.  
- `Raft_Game/`, `Snake_Game/`: 각 게임의 자산 폴더입니다.
